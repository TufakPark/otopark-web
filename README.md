# TufakPark

To install node_modules for client and server

```sh
npm run install:all
```

To install node_modules for client

```sh
npm run install:client
```

To install node_modules for server

```sh
npm run install:server
```

To run front-end

```sh
npm run dev:client
```

To run back-end

```sh
npm run dev:server
```

To run project

```sh
npm run dev:all
```
